# スタンプ工房

LINEスタンプ画像を、申請サイズへ一括変換できるWebアプリです。画像はサーバーへ送信されず、端末のブラウザ内で処理されます。

現在の作業版: `v1.3.1`

保存済み完成版: `v1.0`

## ファイル構成

```text
stamp-koubou/
├── index.html            # 画面のHTML
├── style.css             # 見た目のCSS
├── app.js                # 画像変換、透明化、ZIP作成のJavaScript
├── manifest.json         # PWA用のアプリ情報
├── service-worker.js     # オフライン起動用のキャッシュ処理
├── VERSION.md            # バージョン記録
├── icons/                # ホーム画面用アイコン
├── start-server.command  # iPhone用サーバー起動
├── stop-server.command   # iPhone用サーバー停止
├── make_qr_png.py        # 起動案内ページ用のQRコード生成
└── README.md             # 使い方
```

## 使い方

1. `index.html` をブラウザで開きます。
2. 「スタンプセット」または「絵文字セット」を選びます。
3. 必要なら「ZIPファイル名」にセット名を入力します。
4. 必要な画像枠に画像を追加します。

スタンプセットの場合:
- スタンプ画像: 複数枚、`370×320px`、`01.png`, `02.png`...
- メイン画像: 1枚、`240×240px`、`main.png`
- トークルームタブ画像: 1枚、`96×74px`、`tab.png`

絵文字セットの場合:
- 絵文字画像: 複数枚、`180×180px`、`001.png`, `002.png`... の3桁連番
- トークルームタブ画像: 1枚、`96×74px`、`tab.png`

5. 背景色と透明化の強さを調整します。
6. 必要なら画像ごとに「透明化する」のON/OFFや背景色を調整します。
7. 必要なら「自動プレビュー」をオンにします。
8. 「一括変換する」を押します。
9. 変換後プレビューと申請チェックを確認します。
10. 「申請用ZIP保存」を押して、まとめて保存します。

補助機能:
- 画面上部の「使い方」ボタンから、アプリ内で使い方と注意点を確認できます。
- メイン画像やタブ画像は、1枚目のスタンプ/絵文字画像から自動作成できます。
- スタンプ画像と絵文字画像は、あとから追加した画像が末尾に入ります。
- 追加済み画像は、一覧の「削除」ボタンで1枚ずつ外せます。
- 「背景を透明化する」をOFFにすると、元画像の背景を維持したままリサイズできます。
- 「背景色を自動選択」で、先頭画像の四隅から透明化する背景色を設定できます。
- 画像ごとに透明化ON/OFF、背景色の自動判定、背景色の手動指定ができます。
- 「リセット」で、選択画像と変換結果だけを消して最初の作業状態に戻せます。
- 変換後の画像は、個別保存、PNG一括保存、選択保存もできます。
- iPhoneでは個別の「保存」ボタンから共有シートを開けます。

## iPhoneから自宅Wi-Fi内で使う方法

MacとiPhoneを同じWi-Fiに接続します。

簡単に使う場合は、次のファイルをダブルクリックします。

```text
start-server.command
```

起動すると、iPhoneのSafariで開くURLが表示されます。同時に `server-guide.html` も開き、URLとQRコードが表示されます。

止めるときは、次のファイルをダブルクリックします。

```text
stop-server.command
```

## GitHub Pagesで無料公開する方法

このアプリは静的ファイルだけで動くため、GitHub Pagesにそのまま置けます。

1. GitHubで新しいリポジトリを作ります。
2. このフォルダ内のファイルをリポジトリのルートに入れます。
   - `index.html`
   - `style.css`
   - `app.js`
   - `manifest.json`
   - `service-worker.js`
   - `icons/`
3. GitHubのリポジトリ画面で `Settings` を開きます。
4. `Pages` を開きます。
5. `Build and deployment` の `Source` を `Deploy from a branch` にします。
6. `Branch` を `main`、フォルダを `/root` にして保存します。
7. 数分待つと、次のようなURLで公開されます。

```text
https://ttrs1031.github.io/stamp-koubou/
```

CSS、JavaScript、PWA用ファイルは相対パスで指定しているため、リポジトリ名のサブパス配下でも動きます。

## iPhoneでホーム画面に追加する方法

1. iPhoneのSafariでGitHub PagesのURLを開きます。
2. 画面下の共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。
4. 名前を確認して「追加」を押します。
5. ホーム画面のアイコンから開くと、アプリ風の画面で起動します。

## iPhoneでPNG・ZIPを保存する時の注意

iPhoneでPNGやZIPを保存する場合は、Safariで開いたあと「ホーム画面に追加」して、ホーム画面アプリとして起動する使い方を推奨します。

Safariで直接開いた場合、環境によって保存ボタンがうまく動かないことがあります。ホーム画面アプリ化した状態では、PNG保存・ZIP保存ができることを確認済みです。

## オフライン利用について

初回だけはインターネット接続が必要です。
一度GitHub Pagesで開くと、アプリ本体のHTML、CSS、JavaScript、PWA設定、アイコンがブラウザにキャッシュされます。
その後はオフラインでも基本画面を開けます。

ただし、変換する画像は端末内から毎回選択してください。ブラウザの安全仕様により、選んだ画像そのものをアプリが勝手に保存しておくことはありません。

## 出力について

- スタンプセットZIP: `line-stamp-application.zip`
- 絵文字セットZIP: `line-emoji-application.zip`
- ZIPファイル名にセット名を入力した場合:
  - スタンプセット: `セット名_stamp.zip`
  - 絵文字セット: `セット名_emoji.zip`
- 選択ZIP保存の場合は `selected-` が付きます。
- すべてPNG形式で出力します。
- 絵文字画像は `001.png`, `002.png` ... の3桁連番でZIP直下に出力します。
- 絵文字画像は透明部分を自動トリミングし、約10pxの透明余白を残します。

## 注意

ブラウザの安全仕様により、写真アプリへ無確認で自動保存することはできません。iPhoneで写真アプリに保存する場合は、共有シートから「画像を保存」を選んでください。

画像処理はブラウザ内のCanvasで行います。画像、APIキー、個人情報を外部サーバーへ送信する処理はありません。
